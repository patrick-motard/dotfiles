# Claude Code Repository Outputs

This directory contains outputs generated by Claude Code, organized by repository and branch.

## Structure

```
~/.claude/repos/
  <repo-name>/
    <branch-name>/
      pr-summary/     - PR summary documents
      git-rewrite/    - Git history reorganization files
      <other-types>/  - Future output types as needed
```

## Output Types

### pr-summary/
Contains PR summary documents generated by the pull-request skill.
- `summary.md` - The PR summary document

### git-rewrite/
Contains files from git history reorganization operations.
- `git-history-reorganization-plan.md` - Original detailed plan
- `reorganization-summary.md` - Completion summary
- `commit-msg-*.txt` - Prepared commit messages
- `*-script.sh` - Rebase scripts used
- `*-diff.txt` - Diffs for verification

## Benefits

- **Centralized**: All Claude Code outputs in one location
- **Organized**: By repo and branch for easy navigation
- **Persistent**: Not tied to specific git working directories
- **Flexible**: Easy to add new output types as needed
- **Clean**: Keeps repositories free of generated files
